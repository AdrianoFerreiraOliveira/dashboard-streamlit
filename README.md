Get Started:

## Running locally

Clone the project

```bash
  git clone [https://link-para-o-projeto](https://github.com/AdrianoFerreiraOliveira/dashboard-streamlit.git)
```

Enter the project directory

```bash
  cd my-project
```

Install the dependencies

```bash
  pip install streamlit
```

Start the server

```bash
  python app.py streamlit run
```
